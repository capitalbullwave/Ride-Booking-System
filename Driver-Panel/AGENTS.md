# WaveGo Design System (Driver App)

Brand tokens used across the Flutter driver app. Match these when building or redesigning screens.

## Colors

| Token | Hex | Usage |
|-------|-----|--------|
| Primary | `#31526E` | Buttons, headings, key actions |
| Secondary | `#D8B39F` | Accents, icon backgrounds |
| Background | `#FAF8F4` | Page background |
| Foreground | `#20242C` | Body text |
| Muted | `#E8E4DD` | Borders, chips |
| Muted foreground | `#6086A8` | Secondary text |
| Success | `#5FA87A` | Online, completed |
| Warning | `#E8A95A` | Pending, alerts |
| Error | `#D66B6B` | Errors, SOS |

Defined in `lib/core/theme/app_colors.dart`.

## Typography

- **Headings**: Inter bold (Satoshi when font files added to `assets/fonts/`)
- **Body**: Inter
- See `lib/core/theme/app_typography.dart`

## Radius

- Buttons: 16px (`AppRadius.button`)
- Cards: 20px (`AppRadius.card`)
- Inputs: 18px (`AppRadius.input`)

## App name

**WaveGo Captain** — rides + ambulance mobility platform.
