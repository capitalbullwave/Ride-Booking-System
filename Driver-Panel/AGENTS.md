# Fast Bull Design System (Driver App)

Brand tokens used across the Flutter driver app. Match these when building or redesigning screens.

## Colors

| Token | Hex | Usage |
|-------|-----|--------|
| Primary | `#73398f` | Buttons, headings, key actions |
| Secondary | `#c45cf7` | Accents, icon backgrounds |
| Background | `#ffffff` | Page background |
| Foreground | `#73398f` | Body text |
| Muted | `#f5f0ff` | Borders, chips |
| Muted foreground | `#a47eb6` | Secondary text |
| Success | `#5FA87A` | Online, completed |
| Warning | `#E8A95A` | Pending, alerts |
| Error | `#D66B6B` | Errors, SOS |

Defined in `lib/core/theme/app_colors.dart`.

## Typography

- **Headings**: Satoshi bold
- **Body**: Inter
- See `lib/core/theme/app_typography.dart`

## Radius

- Buttons: 16px (`AppRadius.button`)
- Cards: 20px (`AppRadius.card`)
- Inputs: 18px (`AppRadius.input`)

## App name

**Fast Bull Captain** — rides + ambulance mobility platform.
